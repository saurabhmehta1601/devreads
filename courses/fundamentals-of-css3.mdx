## What is CSS

CSS is the language we use to style an HTML document . CSS describes how HTML elements should be displayed.

</Section>

<Section>

## Example Code

```css
body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}
```
